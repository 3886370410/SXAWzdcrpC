# 医院管理系统简介 python1146

## 项目概述

医院管理系统是一款基于Vue+Django技术栈，采用SQLite作为数据库的软件。该项目实现了前后端分离，确保了系统的高效运行与可维护性。

## 技术栈

- 前端：Vue3+
- 后端：Django
- 数据库：SQLite
- 前端技术：Element-Plus、Axios、Vue-Router、Pinia

## 系统角色

- 管理员
- 医生
- 患者

## 运行环境

- Node.js
- Python

## 部署步骤

1. 确保系统中已安装Node.js与Python环境。
2. 安装前后端依赖。
3. 配置数据库SQLite。
4. 运行后端服务。
5. 前端项目打包部署。

## 目录结构

```
- backend  // 后端Django项目目录
- frontend // 前端Vue项目目录
```

## 功能模块

系统包括但不限于以下功能模块：

- 用户管理
- 病历管理
- 预约挂号
- 药品管理
- 患者信息管理
- 医生排班

## 核心亮点

- 前后端分离，便于大型项目协作开发。
- 采用Element-Plus组件库，界面美观易用。
- 系统模块化设计，易于扩展与维护。
- 支持多种角色，满足不同用户需求。
